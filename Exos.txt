In JS a variable is an element that contains a value like a box having something that can be changed.
var is used to declare a variable in JS's  early life. But it is not recommended to use now. It's better to use let nowadays.
const is used to declare an element which value doesn't change. These elements are called constants.
Datatype : 
	In coding the type of data is used to determine the kind of output that an element can have:
	string: alphanumerical characters
	integer: numerical characters
	float: floating point numbers
	array: table of data
object: 
	is an element which have many properties or attributes. In JS we use class to create an object. For example if we create an object Said we proceed like this 
	class Said {
		name 
		firstname 
		country 
		age 
		...
	}
	name,firstname,country,age are the properties of Said.
